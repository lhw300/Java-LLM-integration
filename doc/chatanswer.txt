/**
 * 将本次 AI 处理结果序列化为 JSON 字符串，供对外 HTTP/WebSocket 接口使用。
 *
 * ┌─────────────────────────────────────────────────────────────────────┐
 * │                        接口字段说明                                  │
 * ├──────────────────┬──────────┬────────────────────────────────────── │
 * │ 字段名           │ 类型     │ 说明                                  │
 * ├──────────────────┼──────────┼────────────────────────────────────── │
 * │ code             │ int      │ 状态码                                 │
 * │                  │          │  0    = 成功                           │
 * │                  │          │ -1    = 系统错误                       │
 * │                  │          │ -100  = 知识库无结果                   │
 * │                  │          │ -101  = 相似度不足                     │
 * ├──────────────────┼──────────┼────────────────────────────────────── │
 * │ answer           │ string   │ TTS 播报文本，直接送 TTS 引擎           │
 * ├──────────────────┼──────────┼────────────────────────────────────── │
 * │ action           │ string   │ VOIP 层需执行的通话动作                 │
 * │                  │          │  NONE     = 无动作，正常播报 answer     │
 * │                  │          │  TRANSFER = 转接人工客服                │
 * │                  │          │  REPLAY   = 重播上一条 TTS              │
 * │                  │          │  VOL_UP   = 调高音量                   │
 * │                  │          │  VOL_DOWN = 调低音量                   │
 * │                  │          │  HANGUP   = 主动挂断                   │
 * ├──────────────────┼──────────┼────────────────────────────────────── │
 * │ intent           │ string   │ 本轮识别的意图大类                      │
 * │                  │          │  QUERY    = 知识库问答                  │
 * │                  │          │  COMMAND  = 系统指令                   │
 * │                  │          │  INFORM   = 用户陈述背景                │
 * │                  │          │  FEEDBACK = 情感反馈                   │
 * │                  │          │  ACK      = 确认/否认                  │
 * │                  │          │  GREETING = 问候/告别                  │
 * │                  │          │  CHITCHAT = 闲聊                       │
 * ├──────────────────┼──────────┼────────────────────────────────────── │
 * │ sentiment        │ string   │ 情感极性，仅 FEEDBACK 时有意义          │
 * │                  │          │  POSITIVE / NEGATIVE / NEUTRAL         │
 * ├──────────────────┼──────────┼────────────────────────────────────── │
 * │ sub_intent       │ string   │ ACK 子类型，仅 ACK 时有值               │
 * │                  │          │  affirm = 肯定  negate = 否定          │
 * │                  │          │  ack    = 普通确认                     │
 * ├──────────────────┼──────────┼────────────────────────────────────── │
 * │ refined_query    │ string   │ 指代消解后的优化查询句，供质检/日志使用
 * ├──────────────────┼──────────┼────────────────────────────────────── │

 *
 * 示例输出（正常问答）：
 * {
 *   "code": 0,
 *   "answer": "您的初始密码是大写A202101小写b。",
 *   "action": "NONE",
 *   "intent": "QUERY",
 *   "sentiment": null,
 *   "sub_intent": null,
 *   "action_code": null,
 *   "refined_query": "教师初始登录密码是多少？",
 
 * }
 *
 * 示例输出（转人工）：
 * {
 *   "code": 0,
 *   "answer": "正在为您转接人工客服，请稍候。",
 *   "action": "TRANSFER",
 *   "intent": "COMMAND",
 *   "sentiment": null,
 *   "sub_intent": null,
 *   "action_code": "ACTION_TRANSFER",
 *   "refined_query": "帮我转接人工客服",
 
 * }
 */


目前对接用到的参数：
COMMAND： TRANSFER REPLAY HANGUP
其他intent 可选
